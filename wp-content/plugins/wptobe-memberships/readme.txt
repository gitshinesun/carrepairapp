== Description ==

A membership and private messaging plugin. Wptobe helps you build any type of website, with member profiles, points and messaging.


**Membership:**

* User registration and profile integrated into your theme
* Users can have multiple memberships
* Create custom registration and profile fields
* Set which fields display and which are required
* You can change the order the profile and registration fields appear on the profile and registration form by dragging.
 [youtube https://www.youtube.com/watch?v=eQU_8T08-2k]
 [youtube  https://www.youtube.com/watch?v=ZjUhTL9s87o]


**Points:**

* Each user on your websites gets their own point balance
* Award customers points for actions like signing up or writing a product review
 [youtube https://www.youtube.com/watch?v=JRrx0YBVR8Y]

**Messaging:**

* Users can privately message one another!

[youtube https://www.youtube.com/watch?v=Fw_kT4ByPFI]

Plugin Support: If you have any issue with this plugin, please visit the plugin site and post it on the support forum : http://wptobe.com/forum/

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
